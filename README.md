# Saber Coub

A 3D lightsaber fighting game with physics-based combat built with Three.js.

## Features

- **Mouse-controlled lightsaber** - Saber tip follows mouse position
- **Saber clash physics** - Saber-to-saber collision creates sparks and bounce
- **Cuttable enemy** - Hit body parts to sever them with physics
- **First-person movement** - WASD + jump

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look around + Aim saber |
| Shift + Mouse | Saber only (camera stays still) |
| Space | Jump |
| Q / E | Turn left/right |
